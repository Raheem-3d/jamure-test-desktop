# Super Admin Dashboard – Design & Developer Brief

This folder contains the end-to-end product + engineering artifacts for the Super Admin Dashboard. It’s designed for Next.js + Tailwind + Prisma and builds on the project’s existing design system.

Included artifacts
- Design tokens (light/dark) and Tailwind mapping
- Layout spec (header, sidebar, shell)
- Component library (variants, props, a11y notes)
- High-fidelity mockup specs (light & dark) for core screens
- Copy text (microcopy, tooltips, empty states)
- API contract (REST), sample JSON responses
- Prisma schema snippets + seeding/guard notes
- Acceptance tests checklist and example test outlines

Quick start
- Tokens: see `design-tokens.md` and update CSS variables if needed.
- Dev: see `api-contract.md`, `prisma-snippets.prisma`, and `middleware-snippets.ts`.
- QA: validate using `acceptance-tests.md`.
